# 과제1: GCD 복잡도 분석 보고서

## 1. GCD 알고리즘 개요

### 1.1 유클리드 호제법 (Euclidean Algorithm)

유클리드 호제법은  GCD(a, b) = GCD(b, a % b) 를 반복 수행 후 나머지가 0이 되면, 그때의 나누는 수가 GCD입니다.

## 2. 시간복잡도 계산 및 분석

```c
int get_gcd(int a, int b) {
    a = abs(a);          // O(1) - 한 번 실행
    b = abs(b);          // O(1) - 한 번 실행
    if (b == 0) return a; // O(1) - 한 번 비교

    while (b != 0) {      // 반복 횟수에 따라 달라짐
        int temp = a % b;  // O(1) - 나머지 연산
        a = b;             // O(1) - 대입
        b = temp;          // O(1) - 대입
    }
    return a;              // O(1)
}
```

만약 a % b = r 이면, r < b  (나머지는 항상 나누는 수보다 작음)
다음 단계에서 b % r = r' 이면, r' < r
두 단계를 거치면: r' < b/2  (반드시 절반 이하로 줄어듦)

그리고 결국 반복 횟수는 a가 아무리 크더라도 b가 작은 경우 금방 끝나게 되므로, 반복 횟수는 b의 크기에 따라 결정됩니다.

따라서, 최악의 경우에도 반복 횟수는 **$O(\log n)$** 이 됩니다. (단, n = a,b 중 작은 수)
