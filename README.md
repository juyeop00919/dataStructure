# 분수추상자료형 ADT

### 1. 변수
- 'num' (numerator) : 분자 
- 'den' (denominator): 분모
- 'res' (result) : 결과
- 'decLen' 소수점 자리수

### 2. 함수
- 'sim' : 기약분수로 변환
- 'add' : +
- 'minus' : -
- 'multi' : x
- 'divide' : /
- 'strFrac' : A/B 형태의 문자열을 분수로 변환 + 정수인경우엔 분모 1 지정 , 소수인 경우엔 소숫점 자리만큼 곱해 분수로 변환
- 'printFrac' : 분수를 A/B 형태로 출력

### 3. 로직
1.시작
2.99자 이하의 숫자를 입력받음 (크기가 100이라하더라도 입력을 99자에서 자름)
3. 입력 받은 숫자에 / 가 포함되면 분수
4. 입력 받은 숫자에 . 이 포함되면 소수
5. 입력 받은 숫자가 정수인 경우 분모 1로 계산
6. 출력
7. 종료
